# 雪猫写作

## 部署/开发注意事项

### tinymce

#### 汉化

- 前往此地址下载中文汉化包： https://www.tiny.cloud/get-tiny/language-packages/
- 在项目public文件夹下新建tinymce文件夹，将下载后的压缩包解压至此文件夹
- 另将node_modules/tinymce/skins文件夹也复制到public/tinymce里
